## How to install this template
### 1. Clone this package
```
git clone https://github.com/marc-aurele-besner/web3-flexible-ui.git
cd web3-flexible-ui
```
### 2. Install dependencies
With NPM
```
npm install
```
Or with Yarn
```
yarn
```
### 3. Run locally
With NPM
```
npm run start
```
Or with Yarn
```
yarn start
```
